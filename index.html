Case Studies

Inventory Demand Forecasting

Problem: Retail teams need accurate daily demand forecasts per store and product to reduce stockouts and avoid overstock.

Approach: Built an end-to-end ML pipeline (ingestion -> transformation -> training -> evaluation). Engineered calendar features and integrated pricing, promotions, and weather signals. Tracked experiments with MLflow.

Serving: Deployed predictions via FastAPI and an interactive Streamlit UI. Added SHAP explainability for transparency.

Results: Produced strong forecast accuracy and generated evaluation reports by store, product, category, and region.

Tech Stack: Python, pandas, scikit-learn, FastAPI, Streamlit, MLflow, Docker, GitHub Actions.

Loan Default Prediction

Problem: Financial institutions must assess loan risk accurately to reduce defaults while maintaining fair approvals.

Approach: Built a modular ML pipeline for data ingestion, feature engineering, and model training. Evaluated models with robust metrics and feature importance checks.

Serving: Exposed predictions via API with a simple UI for testing and stakeholder demos. Automated checks with CI.

Results: Improved risk visibility and delivered a reproducible pipeline ready for deployment.

Tech Stack: Python, scikit-learn, FastAPI, Docker, GitHub Actions.
